# Facing an Issue? 🛠️

If you are facing any issues or need support, here are some resources that might help you.


## How to Report an Issue 📝

1. **Check Existing Issues:** Before reporting, please check the [GitHub Issues](https://github.com/deepesh611/omnilangdb/issues) page to see if someone has already reported the same issue.
2. **Provide Detailed Information:** When reporting an issue, provide as much detail as possible. Include steps to reproduce, screenshots, and any relevant code snippets.
3. **Use Appropriate Labels:** Label your issue appropriately to help us categorize and prioritize it.

## Need More Help? 🙋

If the above resources do not solve your issue, feel free to reach out to our community on [Discord](https://discordapp.com/invite/docusaurus) or ask questions on [Stack Overflow](https://stackoverflow.com/questions/tagged/docusaurus).

We are here to help you! 😊
